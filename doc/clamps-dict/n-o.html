N - O

n-exp

Function 

(n-exp x min max)

Return the exponential interpolation for a normalized value in the range [min..max] as a float value.

Arguments

x An input value in the range [0..1] to be interpolated.

min The output value for x = 0.

max The output value for x = 1.

Examples

(n-exp 0 1 100) ; => 1.0

(n-exp 0.5 1 100) ; => 10.0

(n-exp 1 1 100) ; => 100.0
