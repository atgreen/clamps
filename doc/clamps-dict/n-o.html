n-lin

Function 

(n-lin x min max)

Return a value linearly interpolated between min and max for a given x in the range [0..1].

Arguments:

x Number in the range [0..1].

min Minimum value.

max Maximum value.

Examples:

(n-lin 0 10 20) ;; => 10.0

(n-lin 0.5 10 20) ;; => 15.0

(n-lin 1 10 20) ;; => 20.0
